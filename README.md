# Teslo API

1. Clonar repositorio
2. ```yarn install```
3. ```cp .env.template .env```
4. Cambiar las variables en ```.env```
5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar la app con
```
yarn start dev
```
